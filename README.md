## 기능 요구 사항

### 도메인

- [x] 쿠폰
- [x] 이름
    - 최대 30자
- [ ] 할인
    - [x] 할인 금액
        - 할인 금액은 1,000원 이상
        - 할인 금액은 10,000원 이하
        - 할인 금액은 500원 단위
    - [ ] 할인율
        - 할인율은 `할인금액 / 최소 주문 금액`, 소수점은 버림
        - 할인율은 3% 이상
        - 할인율은 20% 이하
- [x] 최소 주문 금액
    - 최소 주문 금액은 5,000원 이상
    - 최소 주문 금액은 100,000원 이하
- [ ] 카테고리
    - 패션, 가전, 가구, 식품
- [ ] 발급 기간
    - 시작일은 종료일보다 이전
    - 시작일 00:00:00.000000 부터 발급
    - 종료일 23:59:59.999999 까지 발급
